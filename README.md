\

## Installation

```bash
$ npm install
```

## Running the app

```env
# Database Configuration
DB_HOST=your db host
DB_PORT=your db port
DB_USER=your db user
DB_PASSWORD=your db password
DB_NAME=your db name
DB_SSL=
DB_CONNECTION_LIMIT=10
DB_QUEUE_LIMIT=0
DB_ACQUIRE_TIMEOUT=60000
DB_TIMEOUT=60000
JWT_SECRET=base 64 string
JWT_EXPIRATION=24h
REDIS_HOST=your redis host
REDIS_PORT=your redis port
REDIS_PASSWORD=
REDIS_DB=0
```

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
